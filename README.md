# 🚀 NovaTech Company Ltd - IT Training Academy Website

A modern and fully responsive IT Training Academy website built with **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. This project showcases a clean UI, responsive layout, course listings, admission form, student reviews, and company information.
